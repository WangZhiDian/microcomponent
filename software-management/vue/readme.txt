说明：https://cli.vuejs.org/#getting-started

node： node服务器，运行vue前端服务的服务器，类似运行java的jre
vue-cli：创建vue工程的工具，需要提前安装
npm：管理前端工具包的工具，对版本的联动管理，比较欠缺
yarn：也是管理前端工具包的工具管理器，比npm更优越一些，可以控制版本，类似linux的rpm工具。

引入element-ui
引入vue-router ： https://v3.router.vuejs.org/zh/installation.html
引入vue-router时，在npm查找安装到版本： https://www.npmjs.com/package/vue-router/v/3.6.5
vue2 对应到vue-router版本时3.x.x的版本

上述工具都是运行前端所必须要安装的工具，同时注意版本。


全局安装vue-cli（–global：全局安装）
npm install -g @vue/cli
# OR
yarn global add @vue/cli

vue create my-pro
npm run serve  or  yarn serve
